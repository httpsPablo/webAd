# WebAd

## Descripción

WebAd es un proyecto dedicado a la creación de páginas web. Nuestro objetivo es proporcionar soluciones web personalizadas.

## Características

- Diseño responsivo
- Optimización SEO
- Integración con redes sociales
- Panel de administración fácil de usar
